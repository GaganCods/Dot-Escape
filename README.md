## ğŸ® How to Play

1. **Start Drawing** - Click/tap on the green START circle
2. **Navigate** - Draw a continuous line toward the blue END circle
3. **Avoid Obstacles** - Don't touch any red dots!
4. **Reach the Goal** - Complete the path to the END circle
5. **Beat Your Time** - Try to finish faster for higher scores

### Controls
- **Mouse**: Click and drag from START to END
- **Touch**: Tap and drag from START to END
- **Clear Path**: Reset your current drawing
- **Restart**: Generate a new level layout
